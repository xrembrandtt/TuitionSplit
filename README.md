# TuitionSplit

TuitionSplit is a Stellar Soroban-based project that helps students collect tuition contributions from multiple family members or sponsors in one transparent payment pool.

## Problem
A college student in Manila often receives tuition support from different family members, but payments arrive late, through different channels, and are hard to track. This can cause enrollment delays, confusion, and missed deadlines.

## Goal
Build a simple smart contract that can:
- create a tuition fund
- accept contributions from multiple senders
- track total collected amount
- show remaining balance toward a target
